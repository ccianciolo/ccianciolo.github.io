Dweebs Global

Dweebs Global is a Mobile application for a global mentorship project. It provides the platform for mentors and mentees to connect, chat, and work together. This project is currently in production and is in the Black and White wireframe stage.

Problem

The problem we were tasked with solving was taking a list of features and combining them within an app to be developed for iOS and Android.The app will provide chat functionality for both anonymous mental health chat and mentor to mentee conversations. Other functions requested: calendar, mentor search, mentor rating system, view podcasts, and view articles.

Solution

Our solution to this request was designing an app for iOS and Android that is useful for both users and mentors in the Dweebs Global network.

What I learned

I learned a lot about scope creep in this project. This project was immense in terms of initial scope. The team and I tried to get the stakeholders to prioritize features to design in version one of the app. We broke the project into bite sized pieces for each of us to focus on. During this time we met weekly to discuss our progress and give and receive feedback. This project worked best when our team was working well together, collaborating, and communicating in a timely fashion.
