Dweebs Global

Dweebs Global is a Mobile application for a global mentorship project. It provides the platform for mentors and mentees to connect, chat, and work together. This project is currently in production and is in the Black and White wireframe stage.

Problem

Working with a team of 3 other designers our task is to develop a Mobile application based on the pre-research provided by Dweebs Global. The app in it's minimum available product phase will connect mentors and mentees through a mobile platform. It will provide chat functionality for both anonymous mental health chat and mentor to mentee conversations. Other functions included: calendar, mentor search, mentor rating system, view podcasts, and view articles.

Current Dev Stage

We met with stakeholders the other day and were tasked with adding a few more features to the BW Wires. Mentor search and a rating system was requested. Next step is to some testing.
